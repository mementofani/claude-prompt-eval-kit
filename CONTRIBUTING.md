# Contributing

Thanks for helping improve Claude Prompt Eval Kit.

## Pull request checklist

- Add or update an example spec when changing scoring behavior.
- Run `npm test`.
- Keep examples generic and non-proprietary.
- Prefer measurable rubric checks over subjective wording.

## Good contribution ideas

- More example specs
- Better deterministic scoring heuristics
- JSON Schema validation
- GitHub Action integration
- Documentation for prompt review workflows
